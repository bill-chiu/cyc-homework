About them

In a broad sense, a virtual UP host refers to an UP host who uses an avatar to contribute to a video website, and does not restrict whether it is a virtual person. Measured by this standard, Ami Yamato can be regarded as the world's first virtual YouTuber. However, it only uses an animated image to replace its own image for submission, and the activity does not deviate from the author's true identity, and can be considered a virtual YouTuber in a broad sense.

關於

廣義的虛擬UP主是指以虛擬形象在視頻網站上進行投稿活動的up主，並不對是否為虛擬人設進行限制。以此標準進行衡量， Ami Yamato 可視作世界上第一個虛擬YouTuber。但其僅是以動畫形象替代自身形象進行投稿，而活動並未脫離作者的真實身份，可以認為是廣義的虛擬YouTuber。
